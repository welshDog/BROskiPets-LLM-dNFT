## 🦸 Hero Tier (Legendary + Quantum)

### 🕷️ SpiderEep (Legendary)
- **Role**: Lead Hero
- **Agent Power**: Web-crawl specialist, debug thread tracer
- **Signature Move**: `CrawlAndCatch()` — traces bugs across the entire codebase
- **Lore**: Spun from the first line of HyperCode

### 🐍 VenomEep (Legendary)
- **Role**: Security King
- **Agent Power**: Prompt injection detector, adversarial input neutraliser
- **Signature Move**: `VenomBite()` — injects test payloads to find vulnerabilities
- **Lore**: Born from the EchoLeak incident — never lets data escape

### 👑 QueenEep (Legendary)
- **Role**: Commander
- **Agent Power**: Orchestrates all other EEP agents
- **Signature Move**: `RoyalOrder()` — sends simultaneous commands to all agents
- **Lore**: The brain of the EEPVengers squad

### 🦄 UnicornEep (Quantum)
- **Role**: Dream Weaver
- **Agent Power**: Feature flag magician, impossible-feature builder
- **Signature Move**: `MagicFlag()` — enables features that shouldn't exist yet
- **Lore**: One of three Quantum EEPs. Nobody has seen one in the wild.

### 🐕 WelshDogEep (Quantum)
- **Role**: Creator / Master Architect
- **Agent Power**: Builds and upgrades all other EEPs
- **Signature Move**: `HyperfocusBuild()` — enters hyperfocus and ships in 10x speed
- **Lore**: The OG. Born in S.Wales. Powers the entire EEPVengers universe.
