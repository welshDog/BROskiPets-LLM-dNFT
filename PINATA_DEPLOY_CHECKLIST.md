## Step 2 — Deploy BROskiPet.sol to Base Sepolia (5 min)

```bash
cd contracts

# Set env vars first
export ADMIN_ADDRESS=0x<your MetaMask wallet>
export BACKEND_SIGNER_ADDRESS=0x<from step 1>
export AGENT_ADDRESS=0x<same as BACKEND_SIGNER or separate>
export DEPLOYER_KEY=0x<funded wallet private key — needs ~0.01 ETH Sepolia>
export BASE_SEPOLIA_RPC=https://sepolia.base.org
export ETHERSCAN_API_KEY=<from basescan.org — optional for verification>

# Deploy
forge script script/DeployBROskiPet.s.sol \
  --rpc-url $BASE_SEPOLIA_RPC \
  --private-key $DEPLOYER_KEY \
  --broadcast --verify

# Output will include:
# BROskiPet at: 0xDEF...   ← save as BROSKIPET_CONTRACT_ADDRESS
```

Get free Base Sepolia ETH: https://www.coinbase.com/faucets/base-ethereum-sepolia-faucet

## Step 4 — Create Pinata OpenClaw Agent (5 min)

1. Go to https://agents.pinata.cloud/agents
2. Click **Create Agent**
3. Fill in:
   - **Name**: `broski-pet-evolver`
   - **Language**: Python 3.12
   - **Entrypoint**: `pet_evolver_agent.py`
   - **Requirements file**: `requirements.evolver.txt`

4. Under **Secrets**, add:
```
PINATA_JWT                  = [paste yours]
BROSKIPET_CONTRACT_ADDRESS  = [from step 2]
BASE_SEPOLIA_RPC            = https://sepolia.base.org
BACKEND_SIGNER_PRIVATE_KEY  = [from step 1]
OPENAI_API_KEY              = [yours — or set USE_OLLAMA=true]
IPFS_GATEWAY                = aqua-few-dolphin-310.mypinata.cloud
```

5. Under **Triggers**, add:
   - ✅ **Cron**: `0 3 * * *` (daily 3AM UTC)
   - ✅ **Webhook**: enabled (you'll get a URL like `https://your-agent.pinata.cloud/`)

6. Click **Deploy**
